# CoolGame

Динамичная игра на Python с использованием библиотеки Pygame.

## Особенности
- **Одиночный и многопользовательский режимы**: Играйте сами или с друзьями по сети.
- **Система аккаунтов**: Сохранение прогресса и настроек для разных игроков.
- **Система улучшений**: Прокачивайте своего персонажа во время игры.
- **Встроенные читы**: Для тех, кто хочет протестировать возможности игры или просто развлечься (доступны через менеджер читов).

## Требования
- Python 3.x (протестировано до версии 3.14)
- Pygame

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/BANANCHIKIREAL/CoolGame.git
   ```
2. Установите зависимости:
   ```bash
   pip install pygame
   ```
   *Примечание: Если вы используете Python 3.14+, убедитесь, что установлена совместимая версия pygame.*
3. Запустите игру:
   ```bash
   python main.py
   ```

## Управление
- **WASD / Стрелки**: Передвижение
- **Мышь**: Прицеливание и стрельба
- **ESC**: Пауза / Меню
- **R / Enter**: Рестарт (после поражения)
