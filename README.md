# CoolGame (Neon Arena: Eclipse Protocol)

Динамичный аренный шутер на Python с использованием библиотеки Pygame. Сражайтесь с волнами врагов, прокачивайте персонажа и открывайте новые скины.

## Особенности
- **Динамичный геймплей**: Уклоняйтесь от врагов, используйте рывки и собирайте бонусы.
- **Система прогрессии**: Тратьте собранные монеты в магазине на постоянные улучшения и новые облики (скины).
- **Аккаунты**: Поддержка нескольких игровых профилей со своими сохранениями.
- **Мультиплеер**: Играйте вдвоем по локальной сети.
- **Система читов**: Встроенное меню для тестирования возможностей игры.

## Управление
- **WASD / Стрелки**: Передвижение персонажа.
- **Мышь**: Прицеливание.
- **Левая кнопка мыши (ЛКМ)**: Стрельба.
- **Пробел**: Рывок (Dash) — позволяет быстро переместиться и избежать урона.
- **ESC**: Пауза / Выход в меню.
- **R / Enter**: Быстрый рестарт после поражения.
- **TAB**: Смена аккаунта в главном меню.
- **INSERT**: Открыть/закрыть меню читов.

## Функционал

### Мультиплеер (LAN)
Режим сетевой игры работает через UDP:
1. **Важно:** Оба игрока должны быть подключены к **одной Wi-Fi сети**.
2. **Хост**: Выберите "Host" в меню. Игра создаст сервер на порту `5555`.
3. **Клиент**: Выберите "Join", найдите сервер в списке (авто-обнаружение через Broadcast) или введите локальный IP хоста.
4. В мультиплеере каждый игрок управляет своим персонажем, прогресс (опыт, уровень) синхронизируется.

### Система читов
Если вы хотите протестировать игру или просто повеселиться:
- Нажмите **INSERT** во время игры для открытия меню.
- **Переключаемые (Toggle):** Режим бога (God Mode), Убийство с одного удара (One Hit Kill), Бесконечный опыт, Спидхак, Быстрая стрельба.
- **Действия (Actions):** Максимальные улучшения, Спавн босса, Убить всех врагов, Пропустить волну, +5000 монет, Полное исцеление.
- *Примечание: Использование читов в мультиплеере может привести к ошибкам синхронизации.*

## Технические требования
- **Python**: Версия 3.x (Обязательно **до версии 3.14** включительно).
- **Библиотеки**: Указаны в `requirements.txt` (основная — `pygame`).
- **Разрешение экрана**: Оптимизировано под 1400x800.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/BANANCHIKIREAL/CoolGame.git
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите игру:
   ```bash
   python main.py
   ```
