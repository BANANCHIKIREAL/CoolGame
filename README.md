# CoolGame

Динамичная игра на Python с использованием библиотеки Pygame.

## Особенности
- **Одиночный и многопользовательский режимы**: Играйте сами или с друзьями по сети.
- **Система аккаунтов**: Сохранение прогресса и настроек для разных игроков.
- **Система улучшений**: Прокачивайте своего персонажа во время игры.
- **Встроенные читы**: Для тех, кто хочет протестировать возможности игры или просто развлечься (доступны через менеджер читов).

## Функционал

### Мультиплеер
Игра поддерживает сетевой режим:
1. Один игрок выбирает режим "Хост" (Host) в меню.
2. Другие игроки выбирают "Присоединиться" (Join) и вводят IP-адрес хоста.
3. Убедитесь, что порт открыт или игроки находятся в одной локальной сети.

### Читы
Для тестирования и фана в игру встроен менеджер читов:
- Менеджер автоматически загружается при запуске, если файлы находятся в папке `cheats/`.
- В игре можно использовать специальные комбинации клавиш (если предусмотрено интерфейсом чита) или активировать их через консоль/меню отладки.
- **Внимание:** Использование читов в мультиплеере может привести к рассинхронизации!

## Требования
- Python 3.x (**Обязательно до версии 3.14**)
- Библиотеки из `requirements.txt` (в основном `pygame`)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/BANANCHIKIREAL/CoolGame.git
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите игру:
   ```bash
   python main.py
   ```

## Управление
- **WASD / Стрелки**: Передвижение
- **Мышь**: Прицеливание и стрельба
- **ESC**: Пауза / Меню
- **R / Enter**: Рестарт (после поражения)
